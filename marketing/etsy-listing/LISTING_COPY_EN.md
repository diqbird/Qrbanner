1. Etsy sale notification → note buyer email + order ID  
2. **Admin → Etsy orders** → Register order  
3. Review → **Approve & send email**  
4. Buyer receives unique link: `https://qrbanner.com/studio/{token}`  

**Not a fixed link** — each buyer gets a unique token.
